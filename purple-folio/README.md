# Purfolio Theme 2026

Custom WordPress theme with editable header, logo, menu and CTA.

## Install
1. Zip the `purple-folio` folder (the folder itself, not its contents), or upload `Purfolio Theme 2026.zip`.
2. In WordPress: **Appearance → Themes → Add New → Upload Theme** → upload the zip → **Activate**.

## Editable pieces (WP admin)

| What          | Where                                                  |
| ------------- | ------------------------------------------------------ |
| Logo image    | Appearance → Customize → Site Identity → Logo          |
| Brand text    | Appearance → Customize → Site Identity → Brand text    |
| Site title / tagline | Appearance → Customize → Site Identity           |
| Primary menu (desktop) | Appearance → Menus → assign to "Primary Menu (Desktop)" |
| Primary menu (mobile)  | Appearance → Menus → assign to "Primary Menu (Mobile)"  |
| Header CTA button      | Appearance → Customize → Header CTA                     |

If no logo is uploaded, the header falls back to the two-part gradient brand text.
If no menu is assigned to a location, the header falls back to Home / Work / About / Blog / Contact page links.

## Requirements
- WordPress 5.9+
- PHP 7.4+
