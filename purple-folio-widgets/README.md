# Purfolio Addon Plugin

Companion Elementor plugin for **Purfolio Theme 2026**. Adds a
"Purfolio" category to the Elementor editor with fully editable widgets
that mirror the theme's home page sections.

## Widgets

| Widget            | Editable in Elementor                                          |
| ----------------- | -------------------------------------------------------------- |
| PF Hero           | Eyebrow, chip, title (3 parts + highlighted word), lead, primary/ghost buttons + links, hero image |
| PF Services       | Eyebrow, title (3 parts), repeater of cards (icon, title, description) |
| PF Projects       | Eyebrow, title, see-all link, columns (2/3), repeater of projects (image, category, title, link) |
| PF Stats          | Repeater of stats (number, suffix, label)                      |
| PF CTA Banner     | Title (3 parts), description, button label + link              |
| PF Contact Info   | Repeater of items (icon, label, value, optional link)          |
| PF Tools          | Editable technology/tool chips                                 |
| PF Timeline       | Editable animated milestone timeline                           |
| PF Blog Posts     | Responsive latest-posts grid                                   |

Every text, image, link, icon and card is editable directly in the Elementor editor — no code required.

## Install
1. Zip the `purple-folio-widgets` folder (the folder itself, not its contents), or upload `Purfolio Addon Plugin.zip`.
2. WP admin → **Plugins → Add New → Upload Plugin** → upload → **Activate**.
3. Elementor must also be installed and active.
4. Open a page with Elementor → search "PF" in the widget panel → drag any widget in.

## Requirements
- WordPress 5.9+
- PHP 7.4+
- Elementor 3.0+
- Purfolio Theme 2026 active (for exact styling; widgets still render on other themes with fallback styles).
